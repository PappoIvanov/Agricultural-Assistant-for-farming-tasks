# Земеделски Бизнес — Маслодайна Роза

**Стопанство:** 10 дка маслодайна роза (Rosa damascena)
**Начало на проекта:** Април 2026

## Агро Асистент

AI асистент базиран на Claude за водене на дневник, изчисляване на концентрации и препоръки за растителна защита.

### Стартиране

```bash
pip install -r requirements.txt
streamlit run app.py
```

### Първоначална настройка

1. Добави API key в `.env` файла: `ANTHROPIC_API_KEY=your_key_here`
2. Попълни параметрите на стопанството в `config.py` (разстояние между редове, ширина на корона, дюзи)
3. Добави координатите на Парцел 2 в `config.py`

### Файлове

| Файл | Описание |
|---|---|
| `app.py` | Streamlit интерфейс — чат, прикачване на снимки |
| `agent.py` | Claude API логика и обработка на инструменти |
| `tools.py` | Метео, запис в дневник, изчисления на концентрации |
| `config.py` | Координати на парцели, параметри на стопанството |
| `.env` | API key (не се споделя) |
